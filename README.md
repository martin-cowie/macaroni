# macaroni
Resolve MAC addresses to their manufacturer. See [the pitch](https://gitpitch.com/martin-cowie/macaroni).

## Platforms

* MacOS v10.11, v10,12
* Linux v2.6 and later

## Building

    mkdir build && cd build && cmake .. && make all test
